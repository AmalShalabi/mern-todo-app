# MERN Todo App

This is a simple Todo application built using the MERN stack, which consists of MongoDB, Express.js, React.js, and Node.js. 

## Installation

1. Clone this repository using `git clone https://github.com/AmalShalabi/mern-todo-app.git`.
2. Navigate to the project directory using `cd mern-todo-app`.

## Usage

1. Install dependencies by running `npm install`.
2. Start the development server by running `npm start`.
3. Open your web browser and navigate to `http://localhost:3000` to use the app.

## Features

- Create, read, update and delete todo items.
- Responsive design for mobile and desktop devices.
